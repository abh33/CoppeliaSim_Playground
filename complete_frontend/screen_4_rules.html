FRUIT PLUCKING ROBOT THEME DESCRIPTION




- Overall outcome:
To design a robot which can autonomously move in an known arena and pluck and deposit ripe fruits from a tree.

- Major Challenges:
1. Designing and developing robotic arm required for plucking.
2. Line following and Odometry.
3. Identifying ripe and unripe fruits using proper sensor integration.
4. Coding the algorithm for navigation and plucking of fruits.

- Technical Stack:
1. Differential Drive robot.
2. Designing and Control of the Robotic arm.
3. Color Detection and Sensing.

- Major Rules: -
1. The robot will first start from the START position.
2. Once the robot starts, intervention is NOT allowed i.e. the robot is fully autonomous.
3. The robot have to follow the printed lines on the arena for moving.
4. The robot have to correctly distinguish between ripe and unripe fruits and plucks and collects the ripe fruits without any harm to the unripe fruits.
5. While moving and plucking fruits the robot should not damage the arena.
6. After plucking all the ripe fruits, the robot have to deposit them in the deposition Zones.
7. The maximum time given for completing the task is 10 minutes.
