  • Building a Robot Arm

    The basic parts can be designed in the coppeliasim software by grouping basic shapes together. (Note- If you want to design a more complex part, you can design it any CAD software and import it into Coppeliasim. Further info on this can be found here.)

    • Add new shapes from-
      Right Click=>Add=>Primitive Shapes

    • Add joints from-
      Right Click=>Add=>Joints

    • Group two shapes from-
      Right Click=>Edit=>Grouping/Merging=>Group Selected Shapes
