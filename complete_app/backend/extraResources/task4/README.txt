task_4_startingArena.ttt is the blank arena which will be given to the users.
task_4_taskdemo.ttt is the full demonstration of this task.
To run, open the task_4_taskdemo.ttt and run the task_4_eval.py file.

Note-
The users must set the string signal named 'end simulation' as 'true' to indicate that they have finished their run. Then the python script will terminate the connection and display the result.
