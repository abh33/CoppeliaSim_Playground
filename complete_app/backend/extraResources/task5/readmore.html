FRUIT PLUCKING ROBOT THEME DESCRIPTION

- Overall outcome:
To design a robot which can autonomously move in an known arena and pluck and deposit ripe fruits from a tree.

- Major Challenges:
1. Preparing the arena.
2. Designing and developing robotic arm required for plucking.
3. Line following and Odometry.
4. Identifying ripe and unripe fruits using proper sensor integration.
5. Coding the algorithm for navigation and plucking of fruits.

- Technical Stack:
1. Firebird-V robot.
2. ATMEL STUDIO 6 - a free software for programming AVR microcontroller.
3. 3D Cad designing of the Robotic arm.

- Major Rules: -
1. The robot will first start from the START position by pressing only a single switch.
2. Once the robot is switched on, human intervention is NOT allowed i.e. the robot is fully autonomous.
3. The robot must be self contained, and not externally operated by wire or by remote radio control during the competition.
4. The robot have to follow the printed lines on the arena for moving.
5. The robot have to correctly distinguish between ripe and unripe fruits and plucks and collects the ripe fruits without any harm to the unripe fruits.
6. While moving and plucking fruits the robot should not damage the arena.
7. After plucking all the ripe fruits, the robot have to deposit them in the deposition Zones.
8. After depositing the fruits the robot must sounds a continuous buzzer.
9. The maximum time given for completing the task is 10 minutes.
10. A run ends and the timer is stopped when either the robot stops and sounds the continuous buzzer or If the maximum time limit for completing the task is reached or If the team needs repositioning but has used both repositioning options of that run.
